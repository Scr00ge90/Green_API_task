# GREEN-API WhatsApp Test Page

Эта веб-страница позволяет тестировать основные методы WhatsApp API от GREEN-API прямо из браузера. Вы можете получать настройки инстанса, отправлять текстовые сообщения и файлы по URL.

## Описание

Проект представляет собой простой интерфейс для взаимодействия с WhatsApp API через API GREEN-API. Он состоит из трех основных файлов:

*   `index.html`: Структура веб-страницы, включая поля для ввода данных и кнопки.
*   `style.css`: Стили для оформления страницы.
*   `script.js`: Логика работы страницы, отправка запросов к API GREEN-API и отображение ответов.

## Функционал

*   **Параметры инстанса:** Ввод `idInstance` и `ApiTokenInstance` для аутентификации.
*   **Методы инстанса:**
    *   `getSettings`: Получение настроек инстанса.
    *   `getStateInstance`: Получение статуса инстанса.
*   **Отправка сообщений:**
    *   `sendMessage`: Отправка текстовых сообщений на указанный номер.
*   **Отправка файлов:**
    *   `sendFileByUrl`: Отправка файлов (изображений, документов и т.д.) по прямой ссылке URL. Имя файла определяется автоматически из URL.
